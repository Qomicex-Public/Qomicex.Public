# 教程概览

本教程系列将指导你从零开始创建各种类型的插件。

## 教程列表

- [Hello World](hello-world.md) - 创建第一个简单的插件
- [主题插件](theme-plugin.md) - 创建自定义主题
- [设置页插件](settings-plugin.md) - 添加插件设置页面
- [服务注入](service-injection.md) - 使用主程序提供的各种服务

## 高级教程

- [配置存储](advanced/config-storage.md) - 持久化插件配置
- [事件处理](advanced/event-handling.md) - 响应应用事件

## 学习路径

```
创建第一个插件
    ↓
了解主题系统
    ↓
添加设置页面
    ↓
使用可用服务
    ↓
学习高级功能
```

## 开始学习

如果你是新手，建议按照以下顺序学习：

1. [Hello World](hello-world.md) - 了解插件基本结构
2. [设置页插件](settings-plugin.md) - 学习创建 UI
3. [服务注入](service-injection.md) - 使用主程序服务
4. [配置存储](advanced/config-storage.md) - 持久化配置
