# 快速开始指南

欢迎来到 Qomicex Plugin SDK 的快速开始指南。本指南将帮助你快速了解插件系统，并创建你的第一个插件。

## 前置条件

在开始之前，请确保你已经安装了以下内容：

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) 或更高版本
- [Visual Studio 2022](https://visualstudio.microsoft.com/) 或 [Visual Studio Code](https://code.visualstudio.com/)
- 最新版本的 Qomicex Launcher

## 插件是什么？

插件是扩展 Qomicex 启动器功能的模块。通过插件，你可以：

- 添加新的用户界面元素
- 修改应用程序的外观（主题）
- 响应应用程序事件
- 与主程序的核心功能交互
